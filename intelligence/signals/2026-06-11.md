- **[Anthropic/MCP]** The **MCP 2026-07-28 Specification Release Candidate** is published (blog dated **May 21, 2026**) — the largest authorization revision since launch. Six SEPs strengthen auth: mandatory issuer (`iss`) validation per RFC 9207 to stop mix-up attacks; OIDC `application_type` declared during DCR (stops CLI/desktop clients defaulting to `web`); credentials now **bind to the issuing AS's `issuer` and require re-registration when resources migrate**; refresh-token guidance for OIDC-style ASs; and clarified scope accumulation during step-up auth + `.well-known` discovery. (Yesterday's run explicitly deferred this pending publication — now confirmed.) — [MCP blog](https://blog.modelcontextprotocol.io/posts/2026-07-28-release-candidate/) — capability: C2, C4, C10 — domain: IAM-for-AI

- **[Microsoft Entra]** Hard deprecation deadline imminent: the existing **agent registry Graph API will stop being supported on June 15, 2026** (4 days out). Agents registered only via the old API and **not re-registered** through the new **Agent 365–powered** registration Graph API (GA'd May 1, 2026) **will stop functioning**. Consolidates agent management under Agent 365 as the single source of truth; does not affect agents built via Copilot Studio / Foundry. — [MC1297981](https://mc.merill.net/message/MC1297981) / [Graph beta agentId overview](https://learn.microsoft.com/en-us/graph/api/resources/agentid-platform-overview?view=graph-rest-beta) — capability: C1 — domain: IAM-for-AI

- **[NSA / MCP]** NSA's AI Security Center released a Cybersecurity Information Sheet, **"Model Context Protocol (MCP): Security Design Considerations for AI-Driven Automation"** (**May 20, 2026**) — first government baseline guidance for production MCP.
