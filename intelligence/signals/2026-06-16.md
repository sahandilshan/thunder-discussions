- **[CrowdStrike]** **NEW VENDOR (not in `TAXONOMY.md`). FRESH-DATED (June 15, 2026, at Identiverse).** CrowdStrike unveiled **Continuous Identity for AI Agents** inside its Falcon Next-Gen Identity Security suite: each agent gets a **cryptographically verifiable identity using the SPIFFE standard** (replacing static API keys), authorization is evaluated **continuously per-action** on agent ownership + caller + device risk posture, **delegation context is preserved across agent-to-agent chains**, and standing privileges are eliminated (JIT grant/revoke). - **[Beyond Identity]** **NEW VENDOR (not in `TAXONOMY.md`). FRESH-DATED (Identiverse 2026, June 15–16).** Beyond Identity launched **Ceros — an "agentic AI trust layer," now in public preview**. Ceros sits **inline between agents and the services they call** (every API call, tool invocation, and request routes through it before reaching its destination) and applies a **five-dimensional identity check** — the agent's operator, how the human behind it authenticated, device compliance, the apps/tools the agent uses, and the agent's own configuration — plus discovery, runtime policy enforcement, and full audit/provenance across agent sessions, conversations, tools, and tasks. — [Beyond Identity Ceros (public preview)](https://www.beyondidentity.com/resource/introducing-ceros-the-agentic-ai-trust-layer-now-open-for-public-preview) / [AI Trust Layer product page](https://www.beyondidentity.com/products/ai-trust-layer) / [Security Boulevard](https://securityboulevard.com/2026/06/beyond-identity-launches-platform-to-secure-ai-agents/) — capability: C1, C2, C3, C4 — domain: IAM-for-AI

- **[Akamai]** **NEW VENDOR (not in `TAXONOMY.md`). FRESH-DATED (June 15, 2026).** Akamai unveiled a unified **Agentic Security Framework** for Bot & Agent Control, connecting identity, observability, trust, and edge enforcement into a single real-time decisioning layer at the edge. Central mechanism is the **"Know Your Agent" protocol** — built with **Skyfire and Experian** — giving agents a standardized way to declare **identity, origin, and intent**, linking each agent to its platform and the specific authorized human it represents. Six pillars incl. verified identity/human attribution, adaptive trust analysis, and **content monetization/value exchange** (agentic commerce angle). Ecosystem partners: **Visa, Experian, Skyfire, Auth0, Ping Identity, TollBit**. Pushes agent identity into the **CDN/edge + commerce layer**. — [Akamai newsroom](https://www.akamai.com/newsroom/press-release/akamai-unveils-agentic-security-framework-to-power-trusted-ai-driven-interactions-and-commerce) / [GlobeNewswire (June 15)](https://www.globenewswire.com/news-release/2026/06/15/3311619/0/en/Akamai-Unveils-Agentic-Security-Framework-to-Power-Trusted-AI-Driven-Interactions-and-Commerce.html) — capability: C1, C2, C5, C9 — domain: IAM-for-AI
