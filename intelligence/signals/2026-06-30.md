- **[AWS]** **AWS Context** and **AWS Continuum** — revealed at the **AWS Summit New York (June 17)** and never captured in our prior AWS entries (which only ever logged AgentCore Identity's Secrets Manager secret-ARN support). **AWS Context** (announced, "coming soon") is an **identity-aware knowledge graph for every agent in the org**: it auto-builds a graph from existing data (databases, documents, Slack, email) and — the on-taxonomy hook — **each agent query inherits the calling user's IAM + Lake Formation permissions, so an agent acting for a user cannot see or traverse what that user cannot** (delegated, least-privilege agent authorization). **AWS Continuum** is an AI-native security service that "continuously discovers, prioritizes, validates, and remediates" risks, starting in a supervised "learn mode" and earning autonomy category-by-category as customers grant permission. **Dated June 17 — a backfill, not a June 29–30 drop** (flagged in Scout notes).
